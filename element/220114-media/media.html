"Video" 요소 설명

예시

cat1.mp4

속성들
<video> 태그
  • autoplay : 웹페이지 시작 시 자동재생
  • contrlos : 비디오 컨트롤을 표시하도록 지정합니다(예: 재생/일시 중지 단추).
  • loop : 반복 재생
  • muted : 음소거
  • poster : 썸네일
  • preload
    auto : 페이지가 로드될 때 브라우저가 전체 비디오를 로드
    (비디오 파일도 같이 로드된다.)
    metadata : 페이지가 로드될 때 브라우저가 메타데이터만 로드
    (비디오 메타데이터만 로드)
    none : 페이지가 로드될 때 동영상을 로드하지 않음
    (비디오 파일을 로드하지 않음. 재생하기 전 비디오 시간 표시 안됨)
    ※ "autoplay" 속성 사용 시 "preload" 속성은 무시된다.
  • src : 비디오 경로
<source> 태그
<track> 태그
