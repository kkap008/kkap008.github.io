  • h1제목태그요소는 웹페이지를 대표하는 로고에 주로 많이 작성하며, 그밖에 문단내용을 대표하는 제목은 h2-h6을 사용한다.
  • 제목태그는 자동으로 줄바꿈이 일어나는 태그로서 '블럭요소'라고 한다. - 제목태그는 h1, h2, h3, 순으로 작성하며, h2, h1순으로 작성하지 않는다.

html태그학습_02문단태그

-문단관련 태그에는 p, hr, br태그의 종류가 있다.

p : paragraph의 약자로서 내용을 작성할 때 사용하며, 자동으로 줄바꿈이 일어나는 특징이 있다.

hr : horizantal의 약자로서 수평선을 의마하며, 웹페이지의 문단을 구분하고자 할 때 영역과 영역을 구분하고자 할때

pre : HTML에 작성한 내용 그대로 표현합니다. 텍스트는 보통 고정폭 글꼴을 사용해 렌더링하고, 요소 내 공백문자를 그대로 유지합니다.

br : 문단의 줄을 강제로 변경하고자 할 때는 line break인 'br'태그를 사용한다.


html태그학습_03글자관련태그

  1. b : bold의 약자로서 글자를 진하게설정할 때 사용한다.
  2. strong : 글자를 진하게 하고자 할 때 사용하며, b태그와 같은 성격이지만 strong태그는 글자를 강조하고자 할 때 사용하는게 b태그와 차이점이다. 시각장애인이 스크린리더(화면낭동기)프로그램을 사용하여 웹페이지를 이용할 때 strong태그는 강한어조의 음성으로 읽어주는 특징이 있다.
  3. i : italic의 약자로서 글자를 기울이고자할 때 사용.
  4. em : emphasized 의 약자로서 글자를 기울이고자 할 때 사용하며, 강조의 의미로 strong태그요소처럼 강한어조로 읽어준다.
  5. ins : insert의 약자로서 문단내용 중 수정된 글자가 있는 경우글자에 밑줄로 표시하는 태그요소.
  6. del : 글자에 취소선 긋기.
  7. small : 글자의 원래 크기보다 3/4크기로만나타냄.
  8. sup : superscript의 약자로서 위첨자를 나타냄.
  9. sub : subscript의 약자로서 아래첨자를 나타냄.
  10. mark : 문단내용중에 사용자가 지정한 범위에 마킹을 한것처럼 배경색이 들어가는 태그이며, 색상은 노랑색이 기본.
  11. address : 주소의 의미로서 보통 웹페이지의 하단 (footer, 푸터영역)에 회사정보, 저작권정보 등이 작성되는 태그.
