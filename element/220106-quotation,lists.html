quotation

인용문태그는 다른 사람의 말이나 글을 대신하여 인용할 때 사용된다.

blockquote 태그

blockquote

"blockquote" 태그는 자동줄바꿈이 일어나며, 인용부호가 붙지 않는다.

blockquote = block 레벨 요소 = 너비 100%, 높이 콘텐츠 크기 만큼 = 자동 줄바꿈

q 태그

q 태그는 줄바꿈이 일어나지 않으며, 인용부호가 붙는다.

q = inline 레벨 요소 = 콘텐츠 너비,높이만큼 영역 차지 = 자동 줄바꿈 X

그 외 여태까지 사용한 요소 중에는?

p = block 레벨 요소 = 예외사항: 내부에 다른 블록요소 사용불가

br = 인라인 레벨 요소
