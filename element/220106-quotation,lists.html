quotation, lists

quotation

인용문태그는 다른 사람의 말이나 글을 대신하여 인용할 때 사용된다.

blockquote 태그

blockquote

"blockquote" 태그는 자동줄바꿈이 일어나며, 인용부호가 붙지 않는다.
"cite" 속성을 이용하여 출처를 표시

blockquote = block 레벨 요소 = 너비 100%, 높이 콘텐츠 크기 만큼 = 자동 줄바꿈

q 태그

q 태그는 줄바꿈이 일어나지 않으며, 인용부호가 붙는다.
"cite" 속성을 이용하여 출처를 표시

q = inline 레벨 요소 = 콘텐츠 너비,높이만큼 영역 차지 = 자동 줄바꿈 X

lists

  1. 정의목록

    용어에 대한 정의를 내릴 때 사용하며 dl(description list), dt(defines a term), dd(defines description)태그로 작성한다.

    dl 태그
    "dl>dt+dd"로 구성
    dl : 영역을 선언
    dt : "defines a term"으로 용어를 정의하다. 쉽게 생각하면 주제이다.
    dd : "defines description"는 주제에 대한 설명이다.

  2. 순차목록

    ol (order list) : "list type"의 특징은 순서가 있는 것을 사용한다.
    ex) 1,2,3,4,5 ... / a,b,c,d,e ... / 그 외 순서가 있는 것들

    1. "ol>li"로 구성
    2. ol (order list) = 부모
    3. li (list item) = 자식

  3. 비순차목록

    ul (unorder list) : "list type"의 특징은 심볼,이미지를 사용한다.
    ex) 기본 심볼 ● / 이미지

    • "ul>li"로 구성
    • ul (unorder list) = 부모
    • li (list item) = 자식
