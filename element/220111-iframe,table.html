Table

"Table"이란?
행과 열로 이루어진 표
태그 흐름
table > caption+(colgroup>col)+(thead>tr>th)+(tbody>tr>td)+(tfoot>tr>td)

Table Element

  • table : 태이블 범위
  • caption : 테이블 제목을 정의

  • 시멘틱(의미) 태그 - 구획

    • thead : 행 - 속성 제목 영역

    • tbody : 행 - 중심 내용

      ※ "tr>td"로 구성 시 "tbody"는 브라우저마다 틀리지만 자동으로 생성
    • tfoot : 행 - 중심 내용 도출(결과)

  • 핵심 요소 - 내용

    tr : 행을 만들 때 사용

    • th : "thead"에서 사용 - 속성 제목 정의

      요소에 scope="col"을 선언함으로써, 해당 칸이 열의 맨 위에 위치함을 설명할 수 있습니다.
      마찬가지로, scope="row"를 추가하면 해당 칸이 행의 맨 앞에 위치함을 설명합니다.

    • td : "tbody,tfoot"에서 사용 - 속성에 해당하는 내용 정의

  • 관리 요소 - 합치기 및 백그라운드 컬러 지정

    colgroup : 테이블 열 그룹

    • col : 테이블의 각 열을 표현

    • 특징

      "colgroup"의 특징은?
      colgroup은 모든 열에 한번에 공통된 css를 주고싶을 때 유용
      스타일은 ""width, background"만 사용 가능
      "span(폭[범위])"은 여기서 다르게 동작 - 스타일 지정 범위

iframe

iframe - 인라인 프레임 요소
현재 문서 안에 다른 HTML 페이지를 삽입합니다.
각각의 브라우징 맥락은 완전한 문서 환경이므로,
페이지에 "iframe"을 추가할 때마다 메모리 및 기타 컴퓨터 자원 사용량이 늘어납니다.
이론상으로는 원하는 만큼 "iframe"을 사용할 수 있지만, 성능 문제가 없는지 확인하세요.
속성 : name / src
※ 그 외 속성이 있으나 브라우저 지원이 차이가 크다. 필요시 찾아서 선택적 사용하며된다.
