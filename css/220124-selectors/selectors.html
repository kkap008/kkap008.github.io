선택자들

전체선택자(All) 
        
          * {
            font-size:2rem;
            /* 모든 태그 적용 */
            /* 명시도 : 0 */
          }
태그선택자(Tag) 
        
          h1{
            font-size:3rem;
            /* h1 태그 전부 적용 */
            /* 명시도 : 1 */
          }
클래스 선택자(Class) 
        
          .title{
            color:red;
            /* title 클래스에 속하는 태그 전부 적용 */
            /* 명시도 : 10 */
          }

          <h1 class="title">h1</h1>
          <h2 class="title">h2</h2>
아이디 선택자(Id) 
        
          #title{
            color:red;
            /* 유일무이 */
            /* 명시도 : 100 */
          }

          <h1 class="title">h1</h1>
          <h2 class="title "id="title">h2</h2>
          <h3 class="title">h3</h3>
복합선택자(Combinator)
※하위 선택자와 자식 선택자
#하위 선택자 
            
              section p {
                color:red;
                /* <section> All Vertical Depth <p> */
                /* 명시도 : 2 */
              }
#자식 선택자 
            
            section > p {
              color:red;
              /* <section> 1 Vertical Depth <p>
              /* 명시도 : 2 */
            }
※인접 형제 선택자와 일반 형제 선택자
#인접 형제 선택자 
            
              ul > li+li {
                color:red;
                /* <li> 1 sibling */
                명시도 : 3
              }
#일반 형제 선택자 
            
              ul > li~li {
                color:red;
                /* <li> All sibling */
                명시도 : 3
              }
