CSS 소개

CSS(=Cascading Style Sheets)란?
  • 스타일 시트 언어
  • 현재 CSS 범위가 매우 커져서, CSS 권고안을 모듈별로 개발하고 공개
    (W3C는 CSS 명세의 버전을 올리지 않고, 대신 주기적으로 CSS 명세 안정판의 스냅샷을 생성하고 있다.)
  • 핵심 키워드 "Cascading", 즉 상속

Inline CSS

<p style="color: red;">inline CSS</p>

Internal CSS

<head>
<style>
  p { color: blue;}
</style>
</head>
<body>
  <p>internal CSS</p>
</body>

External CSS

<head>
<link rel="stylesheet" href="./external.css" />
</head>
<body>
  <p>external CSS</p>
</body>
